# Agentic ESG Advisor for Urban Waste Management

This project is an AI-powered decision-support system designed to help small and medium-sized businesses assess ESG readiness and prioritize low-cost, high-impact actions related to urban waste management and circular economy practices.

## Key Features
- Agentic ESG reasoning (not a chatbot)
- Focus on waste management and sustainability risks
- Cost-aware and explainable recommendations
- Lightweight Streamlit deployment

## Inputs
- Business type
- Business size
- Waste categories
- Energy source
- Waste handling practices

## Technology Stack
- Streamlit (UI)
- Google Gemini (GenAI reasoning)
- Python (Backend logic)

## Disclaimer
This tool provides decision support and awareness only. It does not offer legal or compliance certification.
